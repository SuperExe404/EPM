Elementos Utilizados en la creacion del dispositivo IOT

Placa ESP-32

Es un microcontrolador de bajo costo y energia, cuanta con tecnologia Wifi y Bluetooth de modo dual integrada que permite controlar todo tipo de sensores, modulos y actuadores.

HC-SR04 Sensor de Distancia

Emite pulsos ultrasónicos y luego detecta el eco de estos pulsos para calcular la distancia entre el sensor y un objeto. Es comúnmente utilizado en proyectos de electrónica y robótica para detectar obstáculos, medir distancias y realizar funciones de detección de proximidad.

Leds

Sirven para mostrar si un circuito está encendido, para indicar la actividad de un sensor o un módulo. Los LEDs son versátiles, fáciles de usar y son comunes en una amplia variedad de proyectos con Arduino.

Jumpers

Los jumpers son pequeños cables utilizados para conectar diferentes componentes dentro de un circuito electrónico. Sirven para establecer conexiones entre pines, módulos, sensores y una placa Arduino o en un protoboard.

Expansor Placa ESP-32

Es un dispositivo que amplía el número de pines de entrada/salida disponibles para conectar componentes externos. La ESP32, aunque potente, tiene un número limitado de pines.
